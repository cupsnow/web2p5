# Develop Note

Base project with **webpack5** and **ReactJS**

## Build

Initialize **Node.js** build environment.

```sh
npm install
```

Start developer server and launch browser to view output.

```sh
npm run start
```

Build for redistribution.

> Output directory **./dist**

```sh
npm run build
```
